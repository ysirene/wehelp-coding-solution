## 09.用預設值填滿空字串

[傳送門](https://wehelp.tw/coding/problem/9)

🟢 難易度：入門

---

輸入一個可能包含空字串的字串陣列 / 列表，以及一個預設值，你的函式能把陣列 / 列表中的空字串用預設值取代。

**範例一**

```
輸入範例一：["Hello", "World", ""]、以及預設值 "test"
回傳：["Hello", "World", "test"]
```

**範例二**

```
輸入範例二：["", "ok", ""]、以及預設值 "failed"
回傳：["failed", "ok", "failed"]
```

**範例三**

```
輸入範例三：["no empty"]、以及預設值 "word"
回傳：["no empty"]
```
