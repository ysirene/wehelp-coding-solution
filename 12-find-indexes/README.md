## 12. 找到目標數字所在的多個索引位置

[傳送門](https://wehelp.tw/coding/problem/12)

🟢 難易度：初級

---

輸入一個整數陣列 / 列表和要找的目標數字，你的函式能找到並回傳目標數字所在的零到多個索引位置。

回傳包含多個索引位置的陣列 / 列表，由小到大排列，若目標數字不存在於陣列 / 列表中，回傳空陣列 / 空列表。

**範例一**

```
輸入範例一：[3, 2, 1, 5, 10]、目標數字 1
回傳：[2]
```

**範例二**

```
輸入範例二：[5, 2, 3]、目標數字 4
回傳：[]
```

**範例三**

```
輸入範例三：[-5, 2, -5, 1, -5]，目標數字 -5
回傳：[0, 2, 4]
```
